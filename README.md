# 🃏 Tunisian Chkobba Engine (Core Logic)

An advanced Python-based game engine for **Chkobba**, the most popular traditional card game in Tunisia. This engine is designed to handle the unique Tunisian variants of the game, focusing on strict capture priorities and traditional scoring.

## ✨ Key Features
- **Authentic Tunisian Rules:** Implements the mandatory "Exact Match over Sum Match" priority.
- **Advanced Scoring:** Complete logic for *Karta*, *Denari*, *Sette Bello*, *Scopa*, and the unique Tunisian *Primiera* (7s then 6s hierarchy).
- **OOP Architecture:** Built with Python `dataclasses` and robust class structures for easy integration with GUI frameworks (Kivy, Pygame, or Unity).
- **Automated Simulation:** Includes a built-in simulation mode to verify game logic and scoring accuracy.

## 🛠 Tech Stack
- **Language:** Python 3.10+
- **Modules:** `itertools`, `dataclasses`, `random`.
